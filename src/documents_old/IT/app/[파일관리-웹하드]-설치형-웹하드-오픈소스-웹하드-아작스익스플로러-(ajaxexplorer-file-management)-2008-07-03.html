회사에 웹하드를 설치할 일이 있어서 찾아보다가,



설치도 간편하고 사용도 편리한 오픈소스 소프트웨어를 발견했다.



이름하야.. AjaxExplorer!



윈도우 환경에 익숙하다면 별 무리없이 사용이 가능하다.



대용량 파일은 업로드가 느리고, 현재버젼에서는 아직 업로드 진행막대를 지원하지 않는듯 하니,



대용량 파일은 FTP를 이용하여 공유하는게 좋을것 같다.



It’s a simple and easy web based file management software!!



AjaxExplorer 설치방법

———————————————-


  1. 다운로드
  2. 서버에 업로드
  3. 압축해제
  4. /etc/httpd/conf/httpd.conf에 AjaxExplorer폴더를 alias 처리
  5. AjaxExplorer 폴더와 하위폴더들의 사용자와 그룹을 아파치로 바꿔준다.
  6. 아파치 재구동
  7. 웹에서 인스톨


AjaxExplorer 단축키(shortcuts)

———————————————-

Esc Sign Out

F1 User Manual

F2 System Account

F3 System Cookies

F4 System Setting

F5 Refresh Directory

F6 Toogle Folders

F7 System Failure

Arrow Navigate through files

Delete Delete highlighted files

Ctrl+A Select all

Ctrl+C Copy highlighted files

Ctrl+D Delete highlighted files

Ctrl+S Save file in Text Editor

Ctrl+V Paste copied/moved files

Ctrl+X Move highlighted files

Ctrl/Shift + Arrow

Ctrl/Shift + Mouse Same effect as Windows Explorer

Only happen when your focus is on Address field:

Arrow Down Navigate to next history directory
